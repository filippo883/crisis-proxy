Per dati live su petrolio, benzina, gas, oro, VIX e cambi real-time serve il Cloudflare Worker gratuito.
1. Vai su workers.cloudflare.com → account gratuito
2. Crea nuovo Worker → incolla il contenuto di worker.js
3. Copia l'URL del Worker e incollalo in const WORKER_URL in questo file
4. Ricarica su Netlify — tutto live!
